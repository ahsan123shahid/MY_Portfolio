### Database Tables:
| Table | Columns | Rows |
|-------|---------|------|
| Users | Id, Name, Email, Password | 0+ |
| Subject | SubjectId, Name | 3 |
| Topic | TopicId, SubjectId, TopicName | 149 |
| KnowledgeChunk | ChunkId, SubjectId, TopicName, Week, Difficulty, Type, Content | 848 |
| Quiz | QuizId, Title, SubjectId, etc. | - |
| Question | QuestionId, QuizId, Content, etc. | - |
| Attempt | AttemptId, UserId, QuizId, Score, etc. | - |
| AttemptAnswer | AnswerId, AttemptId, QuestionId, Answer | - |
| ExplainSession | SessionId, UserId, TopicId, etc. | - |

### Data Import Script:
Location: `import_quiz_data.py`
- Reads: `DATABASE-QUIZ-GENERTOR.xlsx`
- Imports unique subjects (3)
- Imports unique topics (149)
- Imports all knowledge chunks (848)
- Shows row counts after import
